# install-jdk8


```SHELL
sudo apt-get update

sudo apt-get install wget

sudo apt-get install openjdk-8-jdk -y
```

```SHELL
sudo update-alternatives --config java
```

>> choose java-8

```SHELL
java -version
```

```SHELL
sudo update-alternatives --config javac
```
>> choose javac-8

```SHELL
javac -version
```
